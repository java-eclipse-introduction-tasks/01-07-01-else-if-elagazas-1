# 01-07-01-else-if-elagazas-1
Else if többágú elágazás
1. feladat (Víz, gőz, jég)  
Írjon programot az alábbi algoritmus alapján:
```
BE: fok
HA fok>100
   KI: "Gőz"
KÜLÖNBEN HA fok>0
   KI: "Víz"
KÜLÖNBEN
   KI: "Jég"
ELÁGAZÁS VÉGE
```
2. feladat (Hőmérséklet)  
Készítsünk programot, amely felajánlja, hogy (C)elsiusból Fahrenheitbe, vagy (F)ahrenheitből Celsiusba váltja át az adatot (’C’ karakter esetén az első, ’F’ karakter esetén a második). Ezután bekér egy hőmérséklet értéket, majd kiszámolja!  
3. feladat (Síknegyed)  
Adott egy tetszőleges pont a koordinátáival, határozzuk meg, hogy melyik síknegyedben van!
